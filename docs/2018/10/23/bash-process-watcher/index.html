运用场景

2. 监测进程是否在运行

利用 kill -0 检测进程是否存在
kill -0 \$pid中的-0表示不发送任何信号给PID对应的进程，但是仍会对变量值PID对应的进程是否存在进行检查，如果\$pid对应的进程存在，则返回0，不存在返回1。
